Genaigrader

Genaigrader is an AI-powered platform for automated batch evaluation of exams using both local and remote language models. It streamlines the grading process and provides analytics. You can select local models (served in this server by Ollama) or connect to remote models (from an external provider, with your own credentials) for evaluation.

Quick Start

  1. Register or log in to your account.
    Login | Sign up
  2. You can download local models and add external models in the Models section. To connect to a remote model, you will need to provide the following data:
    • API URL: The endpoint of the remote model (e.g., https://generativelanguage.googleapis.com/v1beta/openai/).
    • API Key: Your authentication token for the remote service. This is stored in Genaigrader's database, but is not shared with other users.
    • Model Name: The identifier of the model to use.

    Example of a remote model configuration:

    API URL: https://generativelanguage.googleapis.com/v1beta/openai/  
API Key: sk-1234567890abcdef
Model Name: gemini-2.0-flash-lite-001

    Note: Ensure that the remote model is compatible with OpenAI's API. You can get free access to Google's Gemini models, which provide a OpenAI's compatible endpoint, through Google AI Studio.

  3. Upload exam files and define exam details in the Exam section.

    Exam input format: Each exam file must be a plain text file (.txt) with the following structure:
    • Each question statement is one or more lines, followed by the first option (e.g., A)).
    • Options must be listed as A), B), C), etc., each on its own line. You can use numbers or letters (e.g., 1), 2), a), b)) for options.
    • After the last option, leave a blank line, then write the correct option letter (e.g., A or B) on a new line.
    • Questions are separated by a blank line after the correct answer.
    • Example:
      ¿Cuál es la capital de Francia?
A) Madrid
B) París
C) Berlín

a

¿Cuánto es 2+2?
A) 3
B) 4
C) 5

b
    When you upload an exam, it has to be assigned a course. You can create a new course or select an existing one. The course name must be unique. The Manage Courses sections allows you to view, edit, or delete courses. You can also view the list of exams associated with each course.
  4. Navigate to Batch Evaluations to select exams and models for evaluation. You can choose from available models.
  5. Set the number of repetitions and launch the batch evaluation.
  6. Monitor progress and view results in the results table.
  7. Analyze results and download reports as needed. For more details, visit the Analysis section.
